### Sub-components Used

None.

  

### Fields

- Title (small text, not required)
- Description (small text, not required)
- Icon
    - Icon Name
        - Single select dropdown
        - Options are all the eden icons.
        - No default
- Reference
    - Options are None, External Reference, Internal Reference, Modal, and Site Search
    - Defaults is None

  

### Style Tab

- Component Width 
    - Single select dropdown
    - Options are Read (672px), Pop (1024px), Plus (1600px), and Edge (100%)
    - Default is Read
- Vertical Spacing
    - Single select dropdown.
    - Options are None (0px), Unite (16px), Related (32px), Separated Small (64px), Separated Large (128px), 4px, 8px, and 96px
    - Default is Separated Small (64px)
- Hide From In Page Navigation (boolean, default to false)
- Select a size for the image
    - Single select dropdown
    - Options are Smallest (48x48px), Small (64x64px), Medium (96x96px), and Large (128x128px)
    - Default is Medium (96x96)
- Illustration Color
    - Single select dropdown
    - Options are Black, White, Light Gray, Christmas Red, Easter Purple, Blue, Highlight Blue, and Dark Blue.
    - Default is Black
- Background Color
    - Single select dropdown
    - Options are
        - White
        - Light Gray
        - Blue (MS Use Only)
            
        - Dark Blue (MS Use Only)
            
        - Light Yellow (MS Use Only)
            
        - Medium Gray (MS Use Only)
            
        - Dark Gray (MS Use Only)
            
    - Default is White
- Animate this module on scroll? (boolean, default is false)

  

### States

None.

  

### Front End Requirements

When Animation is turned on, the content will appear in order - icon first, then title, then description - when the component enters the viewport.
