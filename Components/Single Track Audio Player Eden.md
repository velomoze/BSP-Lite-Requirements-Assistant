### Sub-components Used

None.

  

### Fields

- Internal Name (plain text, required)
- Album Name (plain text, not required)
- Cover Art (search select)
- Audio Track (search select, required)

  

### Style Tab

- Vertical Spacing
    - Single select dropdown
    - Options are None (0px), Unite (16px), Related (32px), Separated Small (64px), Separated Large (128px), 4px, 8px, and 96px
    - Default is Separated Small (64px)
- Hide Image (boolean, default is false)
- Component Width
    - Single select dropdown
    - Options are Read (672px), Pop (1024px), Plus (1600px), and Edge (100%)
    - Default is Read
- Hide From In Page Navigation (boolean, default is false)
