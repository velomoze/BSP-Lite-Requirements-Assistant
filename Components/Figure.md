### Sub-components

None

  

### Fields

- Title (small text, not required)
- Media (Search Select, not required)
    - Allows the publisher to add an Image, Video, Titan Image, or Titan Video.
- Caption (small text, not required)
    - Populates with the Caption from Titan, but the publisher can override it.
- Credit (small text, not required)
    - Populates with the Credit from Titan, but the publisher can override it.

  

### Style Tab

- Vertical Spacing
    - Single select dropdown
    - Options are None (0px), Unite (16px), Related (32px), Separated Small (64px), Separated Large (128px), 4px, 8px, and 96px
    - Default to Separated Small (64px)
    - Select a size for the media
        - Single select dropdown
        - Options are Small (403px), Medium (538px), Large (672px), and Extra Large (1024px)
        - Default is Small
- Image Aspect Ratio
    - Single select dropdown
    - Options are 4x1 (shortest), 2.4x1 (short), 16x9 (normal), 4x3 (tall), 1x1 (taller), 3x4 (tallest), Original
    - Default is 16x9 (normal) 
- Hide Caption (boolean, default is false
- Hide Credit (boolean, default is false
- Title Alignment
    - Single select dropdown
    - Options are Start, Center, and End
    - Default is Start
- Caption & Credit Alignment
    - Single select dropdown
    - Options are Start, Center, and End
    - Default is Start
- Background Color
    - Single select dropdown. 
    - Options are
        - White
        - Light Gray
        - Blue (MS Use Only)
            
        - Dark Blue (MS Use Only)
            
        - Light Yellow (MS Use Only)
            
        - Medium Gray (MS Use Only)
            
        - Dark Gray (MS Use Only)
            
    - Default is White
- Background Color Top Padding
    
    - Single select dropdown
        
    - Options are 0px, 8px, 16px, 32px, and 64px
        
    - Default value is 32px
        
- Background Color Bottom Padding
    
    - Single select dropdown
        
    - Options are 0px, 8px, 16px, 32px, and 64px
        
    - Default value is 32px 
        

Video Loop Options

- Start Time (plain text, not required)
- End Time (plain text, not required)
- Loop Video (boolean, default to false)
- Mute Video (boolean, default to false)
- Autoplay Video (boolean, default to false)
- Hide Video Controls (boolean, default to false)
