## Internal Reference

### Fields

- Purpose (not used)
- Referenced Item (search select, required)
    - Publisher selects from existing Brightspot articles.

### Style Tab

- Templates
    - Primary Button
    - Secondary Button
    - Standard Link
    - Transparent Button
        - Invert Button Colors (boolean, default is false)
    - Emphasized Link

  

## External Reference

### Fields

- Purpose (not used)
- Link Text (plain text, not required)
- Accessible Label (plain test, not required)
- URL (plain text, required)
- Leaving Domain Message Type
    - Single select dropdown
    - Options are No, Yes, standard legal notice, Yes, approved custom notice, and Yes, approved exception (no noice required)
    - Default is Yes, standard legal notice

### Style Tab

- Templates
    - Primary Button
    - Secondary Button
    - Standard Link
    - Transparent Button
        - Invert Button Colors (boolean, default is false)
    - Emphasized Link

## Site Search

### Fields

- Purpose (not used)
- Link Text (plain text, not required)
- Accessible Label (plain text, not required)
- Site Search 
    - Search select, required
    - Select from Site Search Pages or Search Results. 
- Query (plain text, not required)
- Filters (add)
    - Add Content or Type filters
- Sort
    - Search select, not required
