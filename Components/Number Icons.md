### Sub-components Used

None.

  

### Fields

- Title (plain text, not required)
- Number (plain text, required)

  

### Style Tab

- Circle Filled (boolean, default is false)
- Color 
    - Single select dropdown
    - Options are Blue, Missionary, Christmas, Easter, and Primary Text
    - Default is Primary Text
- Alignment
    - Single select dropdown
    - Options are Start, Center, and End
    - Default is Start
- Title Size
    - Single select dropdown
    - Options are H2, H3, and H4
    - Default is H3
