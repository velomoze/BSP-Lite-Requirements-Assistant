### Fields

- Description (small text, not required)

### Style Tab

- Component Width 
    - Single select dropdown
    - Options are Read (672px), Pop (1024px), Plus (1600px), and Edge (100%)
    - Default is Read
- Vertical Spacing
    - Single select dropdown
    - Options are None (0px), Unite (16px), Related (32px), Separated Small (64px), Separated Large (128px), 4px, 8px, and 96px
    - Default is Separated Small (64px)
- Component Aspect Ratio
    - Single select dropdown
    - Options are 2.4x1 (shortest), 16x9 (short), 4x3 (normal), 1x1 (taller), 3x4 (tallest), and Custom Height
    - Default is 4x3 (normal)
- Maintain Aspect Ratio at mobile width (boolean, default is false)
- Desktop iFrame Height - Custom Aspect Ratio
    - plain text, not required
- Mobile iFrame Height - Custom Aspect Ratio
    - plain text, not required
- Show Scrollbars (boolean, default is false)
- Show iFrame Borders (boolean, default is false)
- Allow iFrame to be viewed in Full Screen (boolean, default is false)
- Title Style
    - Single select dropdown
    - Options are H2 and H3
    - Default is H2
- Heading Text Alignment
    - Single select dropdown
    - Options are Start, Center, End
    - Default is Start
- Hide From In Page Navigation (boolean, default is false)
